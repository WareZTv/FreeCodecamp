Introduction to Python

Python is a high-level, interpreted programming language known for its clear syntax and readability. Designed to be easy to learn and use, Python is widely used in web development, data science, automation, artificial intelligence, and more. Its large standard library and active community make it a powerful tool for both beginners and experienced developers.

Whether you're writing your first script or building complex systems, Python offers a flexible and efficient environment to get things done quickly and cleanly.

Getting Started with Python

To begin coding in Python, all you need is a text editor and the Python interpreter installed on your system.

Here’s your first Python program:

print("Hello, world!")
Variables and Basic Types

In Python, variables are created by assignment. No type declaration is needed.

name = "Alice" # str age = 30 # int height = 1.65 # float is_student = True # bool

Common types:

  • int → integers (42)
  • float → decimals (3.14)
  • str → text ("hello")
  • bool → True / False
  • None → represents "nothing"

Check a variable’s type:

type(name) # class 'str'

Convert types:

int("10"), str(123), float("2.5"), bool("hi")
Control Structures

if / elif / else

x = 10 if x > 0:     print("Positive") elif x == 0:     print("Zero") else:     print("Negative")while

i = 0 while i < 3:     print(i)     i += 1for

for i in range(3):     print(i)break / continue / pass

for i in range(5):     if i == 3:         continue     print(i)
Functions

Functions let you group reusable code into blocks.

Defining a function

def greet(name):     print("Hello", name)

Calling a function

greet("Alice")

Return a value

def square(x):     return x * x result = square(4) # 16

Functions can have any number of parameters, with or without default values:

def greet(name="World"):     print("Hello", name)
Collections (Lists, Tuples, Dictionaries, Sets)

Python offers built-in collection types to store multiple values.

List ([])

Ordered, changeable.

fruits = ["apple", "banana", "cherry"] fruits[0] # "apple" fruits.append("orange")

Tuple (())

Ordered, immutable.

point = (3, 4) x = point[0]

Dictionary ({})

Key–value pairs, unordered (since Python 3.7: ordered).

person = {"name": "Alice", "age": 30} person["name"] # "Alice"

Set (set())

Unordered, no duplicates.

nums = {1, 2, 3, 2} print(nums) # {1, 2, 3}
Input and Output

Output: print()

print("Hello, world!") name = "Alice" print("Hi", name)

input: input()

name = input("Enter your name: ") print("Hello", name)

input() always returns a string.

Convert input to number

age = int(input("Your age: "))
Modules and Libraries

Modules let you organize and reuse code. Python includes many built-in modules.

Import a module

import math print(math.sqrt(16)) # 4.0

Import specific parts

from math import pi print(pi) # 3.141592...

Install external libraries

Use pip in your terminal:

pip install requests

Then use it in your code:

import requests
Error and Exception Handling

Use try / except to catch and handle errors.

Basic try/except

try:     x = 10 / 0 except ZeroDivisionError:     print("Cannot divide by zero")

Catch all errors

try:     do_something() except Exception as e:     print("Error:", e)

Optional else and finally

try:     x = int("42") except ValueError:     print("Invalid number") else:     print("Success") finally:     print("Done")
Object-Oriented Programming (OOP)

Python supports classes and objects to model real-world entities.

Define a class

class Person:     def __init__(self, name):         self.name = name     def greet(self):         print("Hello,", self.name)

Create an object

p = Person("Alice") p.greet() # Hello, Alice

Inheritance

class Student(Person):     def study(self):         print(self.name, "is studying")

OOP helps organize large programs into reusable and logical components.
